# 🎨 Bike Color Changer

A simple and fun frontend project built using **HTML, CSS, and JavaScript** that allows users to change the color of a bike image with a single click.

This project was inspired by a real-life moment when I visited the **Triumph showroom** with my brother. They didn’t have all the color variants available, so I thought:

**"What if I could visualize how the bike looks in different colors?"**

That idea turned into this mini project 🚀

---

## 📌 Project Overview

The **Bike Color Changer** is a basic interactive web app where users can:

- View a bike image
- Click a button to apply a random color
- Instantly see the bike in a new color

It is a great beginner project to practice:

- DOM manipulation
- Event handling
- CSS styling
- Basic JavaScript logic
- Working with images and visual UI interactions

---

## 🛠️ Tech Stack

- **HTML5** → Structure of the webpage
- **CSS3** → Styling and layout
- **JavaScript (Vanilla JS)** → Functionality and color-changing logic

---

## ✨ Features

- 🎨 Change bike color dynamically
- ⚡ Random color generation on button click
- 🖼️ Clean and minimal UI
- 📱 Beginner-friendly project structure
- 💡 Great for learning DOM manipulation
